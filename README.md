## ⚡️ Usage

1. Get [pnpm](https://github.com/pnpm/pnpm).

   ```bash
   npm i -g pnpm
   ```

1. Install all the dependencies.

   ```bash
   pnpm i
   ```

1. Run local dev server.

   ```bash
   pnpm dev
   ```

1. Clean auto-generated files. (Do not commit those files to the repo.)

   ```bash
   pnpm clean
   ```

## 🚀 Deploy on Vercel

1. Install Vercel CLI.

   ```bash
   npm i -g vercel
   ```

1. Login.

   ```bash
   vercel login
   ```

1. Deploy.

   ```bash
   vercel --prod
   ```
