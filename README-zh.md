## ⚡️ 用法

1. 安裝 [pnpm](https://github.com/pnpm/pnpm)。

   ```bash
   npm i -g pnpm
   ```

1. 安裝所有依賴套件。

   ```bash
   pnpm i
   ```

1. 安裝與設定 Husky。

   ```bash
   pnpm husky
   ```

1. 啟動本地端測試伺服器。

   ```bash
   pnpm dev
   ```
